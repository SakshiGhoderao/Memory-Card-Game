# Memory Card Game

An interactive memory card game built using React.js where users flip cards to find matching emoji pairs.

## Features
- Card flipping animation
- Matching logic with score tracking
- Win condition detection
- Responsive design for different screen sizes
- Dynamic rendering using React components

## Tech Stack
- React.js
- JavaScript (ES6)
- HTML5
- CSS3

## Concepts Used
- useState hook
- Component-based architecture
- Conditional rendering
- Event handling
